What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface used to represent and interact with web documents. It provides a structured representation (like a tree) of a webpage's content, allowing programs (like JavaScript) to read, manipulate, and update the content, structure, and style of the webpage dynamically.

How do you select an element from the DOM?

To select an element from the DOM, you can use various methods in JavaScript. For example:

  • document.getElementById("elementId")
  • document.getElementsByClassName("className")
  • document.getElementsByTagName("tagName")
  • document.querySelector(".className")
What is event delegation in the context of the DOM, and why is it useful?

Event delegation in the DOM is a technique where you attach an event listener to a parent element instead of directly to each individual child element. This means that when an event occurs on any of the child elements, it bubbles up to the parent element, triggering the event listener attached to it.

How do you manipulate an element's attributes and styles using the DOM?

1. Manipulating Attributes

  • let value = element.getAttribute("attributeName");
  • element.setAttribute("attributeName", "value");
  • element.removeAttribute("attributeName");

2. Working with CSS Classes

  • element.classList.add("className");
  • element.classList.remove("className");
  • element.classList.toggle("className");
  • let hasClass = element.classList.contains("className");
